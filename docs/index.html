AES-256-GCM Encryption

Every bookmark title, URL, and folder name is encrypted locally using military-grade encryption. Even if someone extracts your browser storage, they see only ciphertext.

Auto-Lock Timer

Extension automatically locks after 10 minutes of inactivity. Step away from your desk and your bookmarks stay protected. You can also lock manually with the lock icon in the popup or the Alt+L shortcut.

Nested Folders

Create unlimited folder hierarchies with drag & drop support. Rename or delete folders inline. Each folder shows how many bookmarks it contains.

Clear Browsing History

One click removes Chrome's browsing history for all domains in your bookmarks — so no traces are left even outside the extension. Great for privacy.

QR Code

Quick Tab Close Alt+A

Full-text Search

Real-time search through all bookmarks by title or URL directly in the manager. Find anything instantly across all folders — no more endless scrolling.

Encrypted Backup & Restore

Export an encrypted backup file anytime. Restore it on the same or different device — your data stays encrypted throughout. Your password is required to restore.

Bulk Operations

Security in depth

Your master password never leaves your device — and neither does your data

Secure lock screen — the only way in
  • AES-256-GCM encryption

    The same encryption standard used by banks and government systems. Data is encrypted before being written to storage, not after.

  • Auto-lock after inactivity

    Automatically locks after 10 minutes of inactivity. Lock manually anytime with the popup lock icon or Alt+L shortcut (macOS: Cmd+Shift+L). The "Stay Unlocked" setting in Settings disables auto-lock.

  • No password recovery by design

    There is no "forgot password" option — because that would require storing a recovery key, which weakens security. Store your password in a password manager.

Frequently Asked Questions

Quick answers to common questions

How secure is my data?

+

All your bookmarks are encrypted locally using AES-256-GCM encryption with a master password. Your data never leaves your computer. Even we cannot access your bookmarks without your password. The encryption key is derived from your password using PBKDF2 with 600,000 iterations.

What if I forget my password?

+
Password cannot be recovered

We don't store your password or any recovery keys. If you forget your password, you will permanently lose access to all your encrypted bookmarks. Please save your password in a secure password manager like Bitwarden, 1Password, or KeePass.

How do I add bookmarks?

+
  • Click "Add Current Page" in the popup to save the tab you're viewing
  • Click the icon in the popup toolbar to add a bookmark manually (enter URL and title)
  • Right-click any page or link and select "Add to Holy Private Bookmarks"
  • Open the bookmark manager (toolbar icon or Alt+M) and add from there

How do I create folders?

+
  1. Open the extension popup
  2. Click the icon in the popup toolbar to create a new folder
  3. Enter a folder name and confirm
  4. Drag and drop bookmarks into the folder

Can I backup my bookmarks?

+

Yes! Go to Settings → Export & Import → Create Backup. This creates an encrypted backup file. You can restore it anytime using the Restore Backup option.

What does 'Clear History' do?

+

Removes Chrome browsing history for all domains in your bookmarks. This helps maintain privacy by automatically clearing history traces of bookmarked sites. The operation shows a progress bar and can be cancelled at any time.

What does Alt+A do?

+

Alt+A (macOS: Cmd+Shift+A) instantly closes the current browser tab and automatically deletes all browsing history for that domain. A quick shortcut for a discreet exit. Can be reassigned in Chrome at chrome://extensions/shortcuts or via the "Change Shortcuts" button in Settings.

The extension won't unlock

+

If you can't unlock:

  • Make sure you're entering the correct master password (case-sensitive)
  • Try reloading the extension from chrome://extensions
  • If password is forgotten, it cannot be recovered — your data will be lost

Can I import from Chrome?

+

Yes! Open the popup → click the gear icon (Settings) → scroll to the Import from Chrome section. Choose full import (all folders) or selective import (pick specific folders). Your existing bookmarks are preserved — nothing is overwritten.

Can I share a bookmark via QR code?

+

Yes! Hover over any bookmark, click the three-dot icon that appears, and select "QR Code" from the quick actions menu. Scan the generated code with any phone camera to instantly open the URL — no copying or typing required.
